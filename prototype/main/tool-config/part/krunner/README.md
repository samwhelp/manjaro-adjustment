## Usage


### install

run

``` sh
./install.sh
```

or run

``` sh
make install
```


### package-install

run

``` sh
./package-install.sh
```

or run

``` sh
make package-install
```


### asset-install

run

``` sh
./asset-install.sh
```

or run

``` sh
make asset-install
```


### prototype-install

run

``` sh
./prototype-install.sh
```

or run

``` sh
make prototype-install
```


### config-install

run

``` sh
./config-install.sh
```

or run

make config-install
